# 博客系统数据库表结构设计

## 一、数据库概述

| 项目 | 说明 |
|------|------|
| 数据库类型 | MySQL 8.0+ |
| 字符集 | utf8mb4 |
| 排序规则 | utf8mb4_unicode_ci |
| 存储引擎 | InnoDB |

## 三、表结构详情

### 3.1 用户表 (users)

| 字段名 | 类型 | 长度 | 必填 | 默认值 | 说明 |
|--------|------|------|------|--------|------|
| id | BIGINT | - | 是 | AUTO_INCREMENT | 主键，用户ID |
| email | VARCHAR | 255 | 是 | - | 邮箱地址，唯一 |
| password_hash | VARCHAR | 255 | 是 | - | 密码哈希 |
| nickname | VARCHAR | 50 | 是 | - | 用户昵称 |
| avatar | VARCHAR | 500 | 否 | NULL | 头像URL |
| bio | TEXT | - | 否 | NULL | 个人简介 |
| role | ENUM | - | 是 | 'user' | 角色：user/author/admin |
| status | ENUM | - | 是 | 'active' | 状态：active/inactive/banned |
| last_login_at | DATETIME | - | 否 | NULL | 最后登录时间 |
| created_at | DATETIME | - | 是 | CURRENT_TIMESTAMP | 创建时间 |
| updated_at | DATETIME | - | 是 | CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

**索引：**
- PRIMARY KEY (`id`)
- UNIQUE KEY `uk_users_email` (`email`)
- KEY `idx_users_status` (`status`)
- KEY `idx_users_role` (`role`)
- KEY `idx_users_created_at` (`created_at`)

---

| article_count | INT | - | 是 | 0 | 文章数量 |
| created_at | DATETIME | - | 是 | CURRENT_TIMESTAMP | 创建时间 |
| updated_at | DATETIME | - | 是 | CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

**索引：**
- PRIMARY KEY (`id`)
- UNIQUE KEY `uk_categories_slug` (`slug`)
- KEY `idx_categories_parent_id` (`parent_id`)

### 3.4 文章表 (articles)

| 字段名 | 类型 | 长度 | 必填 | 默认值 | 说明 |
|--------|------|------|------|--------|------|
| id | BIGINT | - | 是 | AUTO_INCREMENT | 主键，文章ID |
| title | VARCHAR | 200 | 是 | - | 文章标题 |
| slug | VARCHAR | 200 | 是 | - | URL别名，唯一 |
| summary | VARCHAR | 500 | 否 | NULL | 文章摘要 |
| content | LONGTEXT | - | 是 | - | 文章内容(Markdown) |
| cover_image | VARCHAR | 500 | 否 | NULL | 封面图片URL |
| category_id | INT | - | 是 | - | 分类ID |
| author_id | BIGINT | - | 是 | - | 作者ID |
| status | ENUM | - | 是 | 'draft' | 状态：draft/published/archived |
| is_top | TINYINT | 1 | 是 | 0 | 是否置顶：0否/1是 |
| is_featured | TINYINT | 1 | 是 | 0 | 是否推荐：0否/1是 |
| view_count | INT | - | 是 | 0 | 浏览量 |
| like_count | INT | - | 是 | 0 | 点赞数 |
| comment_count | INT | - | 是 | 0 | 评论数 |
| published_at | DATETIME | - | 否 | NULL | 发布时间 |
| created_at | DATETIME | - | 是 | CURRENT_TIMESTAMP | 创建时间 |
| updated_at | DATETIME | - | 是 | CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

**索引：**
- PRIMARY KEY (`id`)
- UNIQUE KEY `uk_articles_slug` (`slug`)
- KEY `idx_articles_author_id` (`author_id`)
- KEY `idx_articles_category_id` (`category_id`)
- KEY `idx_articles_status` (`status`)
- KEY `idx_articles_is_top` (`is_top`)
- KEY `idx_articles_is_featured` (`is_featured`)
- KEY `idx_articles_published_at` (`published_at`)
- KEY `idx_articles_view_count` (`view_count`)
- KEY `idx_articles_created_at` (`created_at`)
- FULLTEXT KEY `ft_articles_title_summary` (`title`, `summary`)

```
users (1) ──< (N) media

categories (1) ──< (N) articles

articles (1) ──< (N) article_tags
articles (1) ──< (N) comments
articles (1) ──< (N) article_likes
articles (1) ──< (N) visits

tags (1) ──< (N) article_tags

comments (1) ──< (N) comment_likes
```
